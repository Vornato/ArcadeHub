თამაშები

აირჩიე რომელი რეჟიმით გინდა წვეულების დაწყება.

როგორ ერთვებიან მოთამაშეები

ჰოსტი მთავარ ეკრანზე ქმნის ოთახს, მობილურები კი უერთდებიან ოთახის კოდით.

სტანდარტული შეერთება

  1. ჰოსტი ხსნის მთავარ ეკრანს და ქმნის ოთახს.
  2. ეკრანზე გამოჩნდება ოთახის კოდი, LAN მისამართი და QR კოდი.
  3. მოთამაშეები ტელეფონებიდან ხსნიან /play გვერდს.
  4. შეჰყავთ ოთახის კოდი და თავიანთი მეტსახელი.

მოთამაშეები და აუდიტორია

  1. თამაშში ერთდროულად თამაშობს 2-დან 8-მდე ადამიანი.
  2. მეცხრე და შემდეგი შემოსულები აუდიტორიად ერთვებიან.
  3. აუდიტორია პასუხებს არ წერს, მაგრამ ხმის მიცემა შეუძლია.
  4. ლობის ეკრანზე ყველა შეერთებული ადამიანი ცოცხლად ჩანს.

სერვერის ჩართვის შემდეგ ჰოსტის ეკრანი იხსნება http://127.0.0.1:3094/ მისამართზე. ტელეფონებისთვის გამოიყენება http://შენი-LAN-IP:3094/play.

რეჟიმი და წესები

პაკეტი ისეა დალაგებული, რომ საერთო ეკრანზეც მკაფიო იყოს და ტელეფონზეც მარტივად სამართავი.

რას ხედავს ჰოსტი

  1. ოთახის კოდს და შეერთების ინსტრუქციას.
  2. არჩეულ თამაშს, რაუნდის თაიმერს და მოთამაშეთა სიას.
  3. პასუხების გახსნას, ხმის მიცემას, შედეგებს და ქულებს.
  4. შემდეგი ფაზის, პაუზის და მატჩის დასრულების კონტროლს.

რას ხედავს ტელეფონი

  1. პასუხის ჩასაწერ ველს ან ხმის მიცემის ღილაკებს.
  2. ხატვის რეჟიმში პატარა ტილოს, გასუფთავებას და გაუქმებას.
  3. მოლოდინის ეკრანს, როცა სხვა მოთამაშეების ჯერია.
  4. საკუთარ ქულას და სტატუსს ყოველი რაუნდის შემდეგ.
